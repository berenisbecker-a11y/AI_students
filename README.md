# Impact of AI on Students

SQL + BigQuery + Looker Studio analysis of how Generative AI affects academic performance, dependency, anxiety, and skill retention.

---

## Dashboard

![Dashboard](dashboard/dashboard.png)

---

## Project Objectives

- Analyze AI usage patterns.
- Study GPA improvements.
- Evaluate anxiety and dependency.
- Measure skill retention.

---

## Tools

- SQL
- Google BigQuery
- Looker Studio

---

## Key Findings

✔ Advanced prompt skills increase GPA.

✔ AI dependency does not always imply greater AI usage.

✔ Traditional study remains dominant.
